### 📦 Storage App (Production-Level Project)

A full-stack cloud-based file storage application with CI/CD and serverless deployment.

#### 🚀 Key Features:
- Upload & manage files using **Amazon S3**
- Backend deployed on **AWS EC2 & AWS Lambda (serverless)**
- Fast content delivery via **CloudFront CDN**
- Secure authentication + OTP email system
- Redis used for session management
- Automated deployment using **GitHub-based CI/CD pipelines**

#### ⚡ Impact:

- Reduced server load using serverless functions  
- Optimized performance with CDN
